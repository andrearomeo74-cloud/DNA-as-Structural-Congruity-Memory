# Security Policy

## Scope

This repository is a scientific hypothesis framework and not an operational software deployment platform.

Therefore, conventional software security vulnerabilities are generally not applicable.

However, if you identify:

- repository integrity issues
- malicious content
- compromised references
- inappropriate external dependencies
- security concerns affecting repository trustworthiness

please report them responsibly.

---

## Scientific Integrity

The most important form of integrity for this repository is scientific integrity.

Concerns related to:

- falsification errors
- misleading claims
- reproducibility issues
- citation problems
- methodological weaknesses

are welcome through constructive issue reporting.

---

## Responsible Disclosure

Please avoid public reporting of repository integrity concerns before allowing review.
