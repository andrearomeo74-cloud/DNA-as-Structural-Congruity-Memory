# Contributing

Thank you for your interest in this repository.

This project is currently an actively developing hypothesis framework in systems biology and complex systems theory.

At this stage, contributions are welcome primarily in the form of:

- scientific critique
- falsification attempts
- literature suggestions
- methodological improvements
- proxy measurement proposals
- dataset recommendations
- comparative systems biology references

---

## Scope

This repository is not a software product.

It is a scientific hypothesis architecture under active conceptual development.

Constructive skepticism is encouraged.

The objective is not confirmation bias.

The objective is empirical stress testing.

---

## Contribution Principles

Useful contributions include:

### Scientific criticism

Examples:

- identifying logical inconsistencies
- challenging assumptions
- proposing alternative interpretations
- highlighting incompatible literature

---

### Empirical challenge

Examples:

- datasets that may falsify predictions
- biomarkers inconsistent with framework expectations
- resilience evidence contradicting core assumptions

---

### Framework refinement

Examples:

- clearer variable definitions
- improved falsifiability conditions
- stronger mathematical formalization
- better comparative positioning

---

## Out of Scope

This repository is not intended for:

- ideological debates
- non scientific speculation
- design vs evolution polemics
- unsupported metaphysical claims
- promotional contributions

---

## Scientific Standard

A contribution is valuable if it increases clarity, rigor, testability, or falsifiability.

Agreement is not required.

Rigor is.
